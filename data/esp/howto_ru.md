# Gopher-браузер для Sprinter — быстрый старт (ESP Wi-Fi)

Браузер протокола Gopher для компьютера Sprinter, работает под DSS в текстовом
режиме 80×32. Основан на Moon Rabbit / Internet NEXTplorer (автор — nihirash).

Эта сборка использует сетевой бэкенд **ESP Wi-Fi** (SprinterWiFi, ESP8266 /
ESP-AT). Для Wi-Fi нужен сетевой кит SprinterWiFi.

## Что нужно

- Sprinter с Wi-Fi-картой **SprinterWiFi**.
- Установленный **сетевой кит SprinterWiFi** (даёт `NETUP` и `NET.CFG`).
- Файл `GOPHER.EXE` (эта программа) на диске, например в `C:\GOPHER\`.

## Быстрый старт

1. **Настройте Wi-Fi.** Этим занимается сетевой кит SprinterWiFi (его `NET.CFG`);
   как указать свою сеть — смотрите документацию этого пакета.

2. **Поднимите соединение** (один раз за сессию), перед запуском браузера:

       NETUP

   Команда подключается к Wi-Fi и публикует состояние сети, чтобы программы
   могли работать с сетью.

3. **Запустите браузер:**

       GOPHER\GOPHER.EXE

   Он открывается на встроенной домашней странице (сеть не нужна) с несколькими
   стартовыми ссылками. Выберите ссылку и нажмите Enter, чтобы загрузить её.

Если в строке состояния написано `Wi-Fi not up - run NETUP first`, повторите
шаг 2.

## Клавиши

| Клавиша                      | Действие                                       |
|------------------------------|------------------------------------------------|
| Up / Down, Home / End        | перемещение курсора / в начало или конец       |
| PgUp / PgDn (или Left/Right) | страница вверх / вниз                          |
| Enter                        | открыть ссылку / скачать файл / поиск          |
| Backspace                    | назад                                          |
| Esc / F10                    | выход (также прерывает текущую загрузку)        |

Часы (справа вверху в заголовке) берут время из CMOS Sprinter.

## Необязательные файлы рядом с GOPHER.EXE

- **`INDEX.GPH`** — ваша домашняя страница в формате gopher-меню. Если файл есть,
  он заменяет встроенную страницу; редактируйте как угодно.

- **`GOPHER.CFG`** — настройки и сопоставления «открыть скачанный файл в
  программе»:

      [settings]
      skip_ask_for_exec = 0      ; 1/yes = открывать без вопроса, иначе спросить

      [viewers]
      ; ext = программа %file%   (%file% = абсолютный путь к скачанному файлу)
      scr = c:\utils\fview\fview.exe %file%
      gif = c:\bin\gifview.exe %file%

  Когда вы скачиваете файл с указанным расширением, браузер предлагает открыть
  его в сопоставленной программе (`%file%` заменяется путём к сохранённому файлу).

## Загрузки

Бинарные/медиа-элементы (картинки, архивы, образы дисков, …) сохраняются в
каталог `DOWNLOAD\` рядом с `GOPHER.EXE`. Браузер сам их не открывает — только
через сопоставление `[viewers]` выше и только по вашему подтверждению.

## Примечания

- Браузер только открывает TCP-соединения; к Wi-Fi подключается один раз `NETUP`.
