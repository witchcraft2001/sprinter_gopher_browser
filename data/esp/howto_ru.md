# Gopher-браузер для Sprinter — быстрый старт (ESP Wi-Fi)

Браузер протокола Gopher для компьютера Sprinter, работает под DSS в текстовом
режиме 80×32. Основан на Moon Rabbit / Internet NEXTplorer (автор — nihirash).

Эта сборка использует сетевой бэкенд **ESP Wi-Fi** (SprinterWiFi, ESP8266 /
ESP-AT). Для Wi-Fi нужен сетевой кит SprinterWiFi.

## Что нужно

- Sprinter с Wi-Fi-картой **SprinterWiFi**.
- Установленный **сетевой кит SprinterWiFi** (даёт `NETUP` и `NET.CFG`).
- Файл `GOPHER.EXE` (эта программа) на диске, например в `C:\GOPHER\`.

## Быстрый старт

1. **Настройте Wi-Fi.** Этим занимается сетевой кит SprinterWiFi (его `NET.CFG`);
   как указать свою сеть — смотрите документацию этого пакета.

2. **Поднимите соединение** (один раз за сессию), перед запуском браузера:

       NETUP

   Команда подключается к Wi-Fi и публикует состояние сети, чтобы программы
   могли работать с сетью.

3. **Запустите браузер:**

       GOPHER\GOPHER.EXE

   Он открывается на встроенной домашней странице (сеть не нужна) с несколькими
   стартовыми ссылками. Выберите ссылку и нажмите Enter, чтобы загрузить её.

Если в строке состояния написано `Wi-Fi not up - run NETUP first`, повторите
шаг 2.

## Клавиши

| Клавиша                      | Действие                                       |
|------------------------------|------------------------------------------------|
| Up / Down, Home / End        | перемещение курсора / в начало или конец       |
| PgUp / PgDn (или Left/Right) | страница вверх / вниз                          |
| Enter                        | открыть ссылку / скачать файл / поиск          |
| Backspace                    | назад                                          |
| Ctrl+D                       | добавить текущую страницу в закладки           |
| Ctrl+B                       | открыть закладки                               |
| Esc / F10                    | выход (также прерывает текущую загрузку)        |

Часы (справа вверху в заголовке) берут время из CMOS Sprinter.

## Закладки

- **Ctrl+D** — добавить текущую страницу в закладки. Работает только на сетевых
  страницах (у домашней страницы и самого списка закладок адреса нет — их
  закладывать нечего). Браузер дописывает строку в файл `BOOKMARK.GPH` рядом с
  `GOPHER.EXE` (создаёт его при первой закладке).

- **Ctrl+B** — открыть список закладок. Он показывается как обычное gopher-меню:
  курсором выбираете запись, Enter — переход, Backspace — назад. Если закладок
  ещё нет, выводится страница-заглушка с подсказкой.

`BOOKMARK.GPH` — обычный текстовый файл в формате gopher-меню (одна закладка =
одна строка `тип<TAB>название<TAB>селектор<TAB>хост<TAB>порт`), его можно
править вручную или скопировать на другой Sprinter.

## Необязательные файлы рядом с GOPHER.EXE

- **`INDEX.GPH`** — ваша домашняя страница в формате gopher-меню. Если файл есть,
  он заменяет встроенную страницу; редактируйте как угодно.

- **`BOOKMARK.GPH`** — файл закладок (создаётся по Ctrl+D, открывается по Ctrl+B).
  См. раздел «Закладки» выше.

- **`GOPHER.CFG`** — настройки и сопоставления программ. Три секции:

      [settings]
      skip_ask_for_exec = 0      ; 1/yes = открывать без вопроса, иначе спросить

      [viewers]
      ; расширение = программа %file%   (%file% = абсолютный путь к файлу)
      scr = c:\bin\zxview.exe %file%
      txt = c:\utils\fview\fview.exe %file%
      gif = c:\bin\gifview.exe %file%

      [urls]
      ; схема = программа %url%   (%url% = полный URL)
      http  = c:\net\wget.exe %url%
      ftp   = c:\net\ftp.exe %url%

  - **`[viewers]`** — чем открывать **скачанный файл** по его расширению. Когда вы
    скачиваете файл с указанным расширением, браузер предлагает открыть его в
    сопоставленной программе (`%file%` заменяется путём к сохранённому файлу).

  - **`[urls]`** — чем открывать **ссылки на внешние схемы** (gopher-элементы
    типа `h`, обычно `URL:http://…`). Для совпавшей схемы (`http`, `ftp`, …) браузер показывает URL и запускает указанную программу
    (`%url%` заменяется полным адресом). Ссылки `gopher://` браузер открывает сам,
    их в `[urls]` указывать не нужно. Если схемы нет в списке — URL просто
    показывается в строке состояния.

  Имена секций, расширений и схем сравниваются без учёта регистра; путь к
  программе может быть абсолютным или относительным от каталога с `GOPHER.EXE`.
  Подтверждение запуска можно отключить через `skip_ask_for_exec`.

## Загрузки

Бинарные/медиа-элементы (картинки, архивы, образы дисков, …) сохраняются в
каталог `DOWNLOAD\` рядом с `GOPHER.EXE`. Браузер сам их не открывает — только
через сопоставление `[viewers]` выше и только по вашему подтверждению.

## Примечания

- Браузер только открывает TCP-соединения; к Wi-Fi подключается один раз `NETUP`.
